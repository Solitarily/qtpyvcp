class qtpyvcp.plugins.status.StatusItem(item, typ=None, to_str=<type 'str'>, description='')[source]¶

StatusItem object.

Each of the linuxcnc.stat attributes is represented by a StatusItem.

Parameters:
  • item (str) – The name of the linuxcnc.stat item.
  • typ (type) – The python type of the item’s value. If no type is specified the type returned by type(self.value) will be used.
  • to_str (method) – A method which returns a textual version of the item’s value. If not specified defaults to the values __str__ method.
  • description (str) – A human readable description of the item.
