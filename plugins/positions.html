Machine Positions¶

Smart Axis and Join positions plugin.

Calculates Absolute, Relative, and Distance-To-Go values for each axis. Configurable to report actual or commanded positions and values in either machine or program units.

Usage:

Example syntax to use in Widget Rules Editor:

position:axis?abs        # returns tuple of ABS positions
position:axis?rel        # returns tuple of REL positions
position:axis?dtg        # returns tuple of DTG values

class qtpyvcp.plugins.positions.AxisPositions[source]¶

Axis Position Channel.

Axis positions are updated every cycle (~50ms). To save computation resources cached values are returned instead of recalculating for each request.

abs¶

Tuple of ABS axis positions.

dtg¶

Tuple of DTG values for each axis

rel¶

Tuple of REL axis positions.

value¶

Tuple of tuples. REL, ABS and DTG axis positions
